# C0-Compiler

## How to start

For Linux or MacOS with python3:

```sh
# setup
git clone https://github.com/yiranyyu/C0-Compiler
cd C0-Compiler

# get assemble output
./src/cc0.py -s <input-file> -o <output-file>

# get binary output
./src/cc0.py -s <input-file> -o <output-file>

# for more options
./src/cc0.py -h
```

For Windows with python3:

```sh
# setup
git clone https://github.com/yiranyyu/C0-Compiler
cd C0-Compiler

# get assemble output
python ./src/cc0.py -s <input-file> -o <output-file>

# get binary output
python ./src/cc0.py -s <input-file> -o <output-file>

# for more options
python ./src/cc0.py -h
```
